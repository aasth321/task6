#  K-Nearest Neighbors (KNN) on Iris Dataset

##  Overview
This project demonstrates the use of the KNN algorithm for classifying the famous Iris dataset. It covers data preprocessing, model training with different K values, and evaluation using accuracy, confusion matrix, and classification reports.

##  Features
- **Dataset Exploration** – Load and visualize the Iris dataset with Pandas and Seaborn.
- **Data Preprocessing** – Train/test split with stratification and feature scaling using `StandardScaler`.
- **KNN Classifier** – Train models with multiple K values to compare performance.
- **Evaluation Metrics** – Accuracy, confusion matrix, and classification report.
- **Visualization** – Heatmaps and plots for better understanding of results.
